# パーサー設定

パーサーはRSSタイトルから作品名、シーズン、話数、字幕組などを抽出します。

::: tip
v3.1以降、RSSごとのソースパーサーはRSS追加または編集時に設定します。このページではグローバルな有効化、タイトルパーサー、言語、フィルターを設定します。
:::

## WebUI

- **有効化**：RSS解析を有効化します。
- **タイトルパーサー**：`Classic parser (Stable)` は従来の動作を維持します。`Universal parser (Preview)` は話数範囲、OVA、劇場版、混合集を解析します。両エンジンは自動的に相互フォールバックしません。
- **言語**：解析に使う優先言語です。`zh`、`jp`、`en` をサポートします。
- **除外**：グローバルフィルターです。文字列または正規表現を指定できます。

## `config.json`

セクション：`rss_parser`

| キー | 説明 | 型 | WebUI項目 | 既定値 |
| --- | --- | --- | --- | --- |
| `enable` | RSSパーサーを有効化 | 真偽値 | 有効化 | `true` |
| `engine` | タイトルパーサー：`classic` または `tokenizer` | 文字列 | タイトルパーサー | `classic` |
| `filter` | グローバル除外ルール | 文字列配列 | 除外 | `["720", "\\d+-\\d+"]` |
| `language` | 解析言語 | 文字列 | 言語 | `zh` |
